# Data Layer - Raw Data
This folder contains raw data from original sources.

## Structure
```
raw/
├── demographics/     # Population, census data
├── economic/         # GDP, employment, sectors
├── environment/      # Land cover, water, climate
├── infrastructure/   # Transport, utilities, facilities
├── projects/         # Mega-projects, investments
└── administrative/   # Boundaries, admin units
```

## Guidelines
- **DO NOT** modify raw data files
- Keep original filenames with date suffix
- Document source and download date
- Use metadata/ folder for data dictionaries
