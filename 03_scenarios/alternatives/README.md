# Scenarios Layer - Alternative Scenarios
This folder contains alternative future scenario outputs.

## Scenarios
- CLIMATE_STRESS: Climate stress scenario
- TECH_DISRUPTION: Technology disruption scenario
- ECON_TRANSFORM: Economic transformation scenario
- DEMO_SHIFT: Demographic shift scenario

## Contents per Scenario
- Scenario narrative
- Key drivers and assumptions
- Parameter settings
- Model outputs
- Comparison with baseline
- Risk/opportunity assessment
